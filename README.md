# distroless-python3-machinelearning

A minimal Docker image based on [`gcr.io/distroless/python3`](https://github.com/GoogleCloudPlatform/distroless/tree/master/python3) that comes with:

- pandas
- numpy
- scikit-learn
- matplotlib
- scipy
- six
- gfortran
